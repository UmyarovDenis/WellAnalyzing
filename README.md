# Инструкция по запуску

1) Скомпилировать приложение
2) Запустить WellAnalyzing.exe


# Начало работы

1) В левом верхнем углу приложения нажать элемент меню **Файл** -> **Импорт** появится диалоговое окно выбора файла
2) Выбрать .csv файл с исходными данными (можно забрать [отсюда](https://github.com/UmyarovDenis/WellAnalyzing/blob/master/wells.csv))
3) Далее начнётся формирование сводок по скважинам.

## Экспорт в JSON

После формирования сводок, можно экспортировать значения в JSON:
1) В левом верхнем углу приложения нажать элемент меню **Файл** -> **Экспорт** появится диалоговое окно сохранения файла
2) Подтвердить сохранение
